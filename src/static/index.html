Bem-vindo ao Aplicativo de Autenticação

Este aplicativo demonstra o fluxo de autenticação com Microsoft Entra ID.

Para iniciar o login, você normalmente receberia um link formatado assim (substitua os placeholders):

/auth/login?whatsapp_id=SEU_WHATSAPP_ID_AQUI

Se a aplicação estiver rodando localmente na porta 5001, o link completo seria:

http://localhost:5001/auth/login?whatsapp_id=SEU_WHATSAPP_ID_AQUI

Clique aqui para testar o login com whatsapp_id=test_whatsapp_id_123 (requer configuração do .env e do App Microsoft Entra ID)
